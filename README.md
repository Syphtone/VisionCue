# VisionCue
VisionCue is an AI powered Interview Coach where candidates can practice job interviews and improve their performance based on AI feedback
This project analyzes **perceived confidence levels** in interview recordings using **multimodal data** combining facial and audio cues. The AI system integrates **OpenFace** for facial landmark-based emotion tracking and **OpenSmile** for acoustic feature extraction. A **Long Short-Term Memory (LSTM)** classifier processes sequential data across frames to evaluate confidence, achieving an accuracy of approximately **76%**
